# 先驱者：游戏介绍

## 1. 一句话概念

《先驱者》是一款以“认知世界”为核心玩法的探索与文明演化游戏。玩家扮演某个种群中率先觉醒认知能力的个体，在一个最初无法理解的世界中，通过观察、尝试、记忆和传播知识，逐步将未知的“马赛克世界”转化为可理解、可利用、可传承的文明基础。

## 2. 核心主题

游戏的主题不是单纯的生存，也不是传统科技树升级，而是：

```text
未知如何被认知，经验如何变成知识，知识如何推动文明。
```

在《先驱者》中，玩家开局几乎没有任何知识。世界中的环境可以被感知，但大多数对象无法被识别，它们在玩家眼中呈现为模糊的马赛克。玩家需要通过不断实验，理解每个对象的性质、风险、用途和组合关系，并将这些认知传播给种群，使整个族群从本能生存逐步走向高级文明。

## 3. 玩家身份

玩家不是传统意义上的英雄，而是种群中的第一个“先驱者”。

先驱者具备特殊能力：

- 能主动观察未知对象。
- 能从尝试中形成记忆。
- 能把个人经验抽象成可传播的知识。
- 能将知识传递给族群，改变族群行为。
- 能带领族群突破生存边界，进入更高文明阶段。

玩家的价值不只在于亲自探索，而在于把探索结果沉淀为群体可以继承的认知资产。

## 4. 世界状态

游戏世界存在两层状态：真实世界与认知世界。

### 4.1 真实世界

真实世界是规则引擎内部的客观状态。每个对象都有固定或动态的真实属性，例如：

- 是否可食用。
- 是否有毒。
- 是否可燃。
- 是否坚硬。
- 是否会攻击。
- 是否可治疗。
- 是否可作为工具材料。
- 是否可以和其他对象组合产生新用途。

真实世界不因为玩家是否理解而改变。玩家不知道火是什么，火依然会燃烧；玩家不知道毒蘑菇有毒，食用后依然会中毒。

### 4.2 认知世界

认知世界是玩家和族群当前理解到的世界。开局时，大部分对象只显示为未知马赛克。

随着玩家实验，认知逐步变清晰：

```text
未知马赛克
  ↓
可观察轮廓
  ↓
部分特征
  ↓
推测名称
  ↓
确认对象
  ↓
族群共享知识
```

认知世界决定玩家界面中看到的名称、图标、提示、风险标记和可用行为。它是玩家理解游戏规则的窗口。

## 5. 核心玩法循环

《先驱者》的核心循环是：

```text
进入未知区域
  ↓
发现马赛克对象
  ↓
选择实验行为
  ↓
承受风险或获得收益
  ↓
获得认知反馈
  ↓
形成个人记忆
  ↓
传播给族群
  ↓
解锁新的群体能力
  ↓
探索更危险的未知区域
```

这个循环的乐趣来自“有线索的试探”，而不是盲目穷举。玩家需要根据观察到的颜色、形态、行为、环境关系和过往记忆做判断。

## 6. 核心抽象层

为了让游戏规则可扩展、可验证、可复盘，《先驱者》的底层设计优先抽象为以下概念。

### 6.1 世界对象 Object

世界对象是游戏中可被感知、交互、认知和传播的基本单位。

对象可以是：

- 植物。
- 动物。
- 矿物。
- 地形。
- 火焰。
- 水源。
- 工具材料。
- 遗迹。
- 天气现象。

每个对象都有真实属性和认知属性。真实属性用于规则结算，认知属性用于玩家理解。

### 6.2 特性 Trait

特性是对象能够被认知的最小语义单位。

例如：

- 可食用。
- 有毒。
- 可燃。
- 坚硬。
- 锋利。
- 温暖。
- 危险。
- 可治疗。
- 可储水。
- 可驱赶野兽。

一个对象可以拥有多个特性。玩家不是一次性知道对象，而是逐步发现特性。

### 6.3 行为 Action

行为是玩家对世界对象发起的实验方式。

第一阶段建议保留少量高价值行为：

- 观察：低风险，获得外观、环境和行为线索。
- 触碰：中风险，获得温度、硬度、材质、刺痛等信息。
- 食用：高风险，验证食物、毒性、治疗等性质。
- 攻击：验证硬度、反击、破坏、掉落和威胁。
- 采集：取得样本，用于后续组合或传播。
- 组合：测试两个对象之间是否产生新用途。
- 传播：将个人认知转化为族群共享知识。

行为不是按钮集合，而是玩家认知世界的方法论。

### 6.4 反馈 Feedback

反馈是行为后产生的可感知结果。

反馈分为四类：

- 生理反馈：饥饿、受伤、中毒、恢复、疲劳。
- 感知反馈：颜色、声音、气味、温度、硬度、运动。
- 环境反馈：燃烧、扩散、吸引野兽、驱赶野兽、污染水源。
- 认知反馈：发现新特性、提升置信度、修正错误认知、形成名称。

反馈必须服务于决策。每次实验都应该让玩家更理解世界，而不是只给随机结果。

### 6.5 记忆 Memory

记忆是先驱者个人层面的认知沉淀。

记忆记录的是“我对某类对象的理解”，包括：

- 已发现特性。
- 特性置信度。
- 已尝试行为。
- 行为结果。
- 风险记录。
- 与其他对象的组合关系。

记忆允许玩家从经验中学习，也允许规则引擎判断一个对象是否已经从马赛克转化为清晰对象。

### 6.6 知识 Knowledge

知识是可传播、可复用、可影响族群行为的稳定认知。

记忆偏个人，知识偏群体。一个经验成为知识，通常需要满足：

- 足够高的置信度。
- 足够明确的对象或特性。
- 能被表达、模仿或记录。
- 能对族群行为产生实际影响。

例如：

```text
“红浆果可食用”是知识。
“我吃过一个红色东西之后不饿了”只是记忆。
```

### 6.7 传播 Propagation

传播是个人认知转化为族群能力的过程。

早期传播依赖模仿和示范，中期依赖符号和语言，后期依赖文字、教育、数据库和网络化意识。

传播结果可以改变族群行为：

- 自动采集安全食物。
- 避免已知毒物。
- 使用火进行防御。
- 使用工具狩猎。
- 储存药草治疗伤员。
- 规划迁徙路线。

### 6.8 文明能力 Civilization Capability

文明能力是知识积累后的系统性解锁。

它不是简单科技点，而是由多个知识共同支撑的群体能力。

例如：

```text
识别树枝 + 理解石头坚硬 + 理解尖锐伤害
= 解锁简易石矛
```

```text
理解火会燃烧 + 理解火能驱兽 + 理解木材可燃
= 解锁夜间营地防御
```

## 7. 规则引擎定位

《先驱者》的规则引擎是整个游戏的权威裁判。它不负责渲染、动画、音效和 UI，只负责状态变化与规则结算。

规则引擎的核心输入输出是：

```text
输入：当前状态 + 玩家行为
输出：新状态 + 事件列表
```

规则引擎需要回答：

- 行为是否合法。
- 对象真实属性是什么。
- 行为触发了哪些结果。
- 玩家获得了哪些认知。
- 认知置信度如何变化。
- 是否形成新记忆。
- 是否能传播为知识。
- 是否解锁文明能力。
- 是否产生危险、死亡、胜利或阶段变化。

## 8. 事件驱动表达

游戏内所有重要结果都应该以事件表达，而不是静默修改状态。

示例事件：

- ActionStarted：行为开始。
- ActionRejected：行为被拒绝。
- ObjectObserved：观察到对象线索。
- TraitDiscovered：发现新特性。
- ConfidenceChanged：认知置信度变化。
- MemoryFormed：形成个人记忆。
- KnowledgeShared：知识被传播。
- ObjectRevealed：对象从马赛克转为清晰。
- HungerChanged：饥饿值变化。
- HealthChanged：生命值变化。
- Poisoned：中毒。
- CapabilityUnlocked：文明能力解锁。
- CivilizationStageChanged：文明阶段变化。

事件既用于前端播放表现，也用于调试、复盘、测试和叙事记录。

## 9. 文明阶段

游戏可以按认知传播能力划分文明阶段。

### 阶段 0：混沌感知

玩家几乎无法分辨对象，只能观察、靠近、逃离和少量尝试。

### 阶段 1：本能生存

玩家开始识别食物、危险、庇护所和简单材料，族群生存率提升。

### 阶段 2：工具认知

玩家理解硬度、锋利、组合关系，开始制造简单工具。

### 阶段 3：火与营地

玩家理解火、夜晚、防御、熟食和温度，族群形成稳定营地。

### 阶段 4：符号文明

知识不再只靠模仿传播，开始通过符号、壁画、语言和记录跨代保存。

### 阶段 5：系统科学

族群开始主动设计实验，理解因果关系、材料规律、医学、农业和建筑。

### 阶段 6：高级文明

知识形成网络化结构，族群能够组织大规模探索、研究未知，并主动改造环境。

## 10. 第一阶段设计边界

当前阶段只设计抽象层，不实现 Demo，不做完整内容量。

第一阶段重点是定义：

- 对象如何被认知。
- 行为如何产生反馈。
- 记忆如何形成。
- 知识如何传播。
- 文明能力如何解锁。
- 规则引擎如何表达状态变化。

当前阶段暂不解决：

- 完整地图生成。
- 复杂 AI。
- 多人同步。
- 美术资源生产。
- 商业化系统。
- 大规模科技树。
- 复杂 UI 动效。

## 11. 最小概念样例

一个早期场景可以这样成立：

```text
玩家在营地附近发现三个未知马赛克：A、B、C。

A 是红浆果。
B 是毒蘑菇。
C 是石头。

玩家观察 A，发现它是红色植物。
玩家食用 A，饥饿下降，获得“可食用”认知。
玩家传播 A 的知识，族群开始采集 A。

玩家观察 B，发现它颜色鲜艳。
玩家食用 B，中毒，获得“有毒”认知。
玩家传播 B 的知识，族群避开 B。

玩家触碰 C，发现它坚硬。
玩家用 C 攻击树枝，发现可以破坏和加工。
玩家形成“坚硬材料”记忆，为工具认知打基础。
```

这个样例证明了游戏最核心的体验：未知对象通过实验变成知识，知识改变族群命运。

## 12. 设计目标

《先驱者》的设计目标是让玩家感受到：

- 世界不是一开始就可理解的。
- 知识来自带风险的尝试。
- 每个对象都有被理解的过程。
- 个体经验可以变成群体文明。
- 文明进步不是数值膨胀，而是认知结构升级。

玩家最终带领种群走向高级文明时，获得的不是单纯的等级提升，而是从混沌中建立秩序、从未知中创造意义的成就感。
