## 8. 生产链路验证

```text
pathfinder_client_server_exe
  -> ClientServerRuntimeFactory
    -> WorldGridRuntime (initialize + generateInitialWorld)
    -> ClientRuntimeProjectionBridge (RuntimeBacked)
    -> ClientRuntimeCommandOptionBridge (RuntimeBacked)
    -> ClientProjectionAdapter (注入 bridge)
    -> ClientAvailableCommandAdapter (注入 bridge)
    -> ClientSessionGateway
    -> ClientCommandGateway
    -> WorldCommandPipeline
      -> runtime-aware Move/Inspect/Wait/GenerateWorld handlers
    -> ClientHttpGateway
```

验证命令（编译并运行）：
```bash
cd build/backend && make -j$(nproc) pathfinder_client_server_exe
./pathfinder_client_server --port 1999
# 另开终端
curl -X POST http://127.0.0.1:1999/api/client/bootstrap \
  -H "Content-Type: application/json" \
  -d '{"client_id":"c1","session_id":"s1","requested_actor_key":"player"}'
```

返回结果应包含非空 `visible_cells`、`visible_entities` 和 `available_commands`（含 Move 选项）。
